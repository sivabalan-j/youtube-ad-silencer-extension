# YouTube Ad Silencer Extension

## Summary

A lightweight Chrome extension that silences YouTube ads, skips skippable ads, and speeds up unskippable ads to minimize interruptions. This extension is intended for **local use only** and must be loaded manually in developer mode.

## Standard Operating Procedure (SOP)

### Prerequisites

* Google Chrome browser
* Extension source code downloaded from the `main` branch

### Installation Steps

1. Download or clone the source code from the **main** branch.
2. Open **Google Chrome**.
3. Go to:
   `Settings` ‚Üí `Extensions`
   or navigate directly to `chrome://extensions/`
4. Enable **Developer Mode** (top-right corner).
5. Click **Load unpacked**.
6. Select the folder containing the extension source code.
7. The extension will now be added to your browser.

### Usage

1. Open **YouTube** in Chrome.
2. Click on the **YouTube Ad Silencer Extension** icon.
3. Enable **Ad Blocking** from the extension interface.
4. Enjoy a quieter and faster YouTube experience.
